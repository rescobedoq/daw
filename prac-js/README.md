# Descripción de la práctica
- Buscar documentación sobre JavaScript.
- Reutilizar los ejemplos y ejercicios encontrados para crear un ejercicio en el que el usuario modifique el DOM mediante eventos.
- Documentar el ejercicio propuesto en la página web de Google Sites creada en la clase de teoría. (Utilizar el identificador del grupo para crear la página)
- Crear un video que muestre la funcionalidad del ejercicio, muestre el Firebug del navegador y evidencie la modificación del DOM mediante eventos desencadenados por el usuario con JS. (Máximo 3 min)

# Entregables
- Página en Google Sites.
- Video (incrustarlo en la misma página web).
